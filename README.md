# Island Environment 🏝️

Progetto di ambiente 3D interattivo realizzato in **Unity**, con modelli 3D creati da me in **Blender**.

## 🛠️ Tecnologie utilizzate
- **Unity 3D**: Motore di gioco utilizzato per la scena e l'interattività.
- **Blender**: Utilizzato per la modellazione 3D e l'esportazione degli asset (.fbx).
- **C#**: Linguaggio usato per gli script di movimento e interazione.

## 📁 Struttura della Repository
- `/Assets/Modelli/Sorgenti_Blender`: Contiene i file `.blend` originali.
- `/Assets/Scripts`: Contiene i file di codice C#.
- `/Assets/Texture`: Contiene i materiali e le texture applicate.

## 🚀 Come avviare il progetto
1. Clona questa repository sul tuo PC.
2. Apri la cartella con **Unity Hub** (versione consigliata: 2023.x o superiore).
3. Apri la scena principale situata in `Assets/Scenes`.
